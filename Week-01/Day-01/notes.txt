1. Introduction
------------------------------------------------------------------------
* Hackers are doers who want to touch, play, create, and sometimes break 
  things, rather than reading long tomes of IT theory.
* This chapter provides fundamental skills to get up and running in Kali Linux 
  just enough to play and explore, saving in-depth discussions for later.


2. Introductory Terms and Concepts
------------------------------------------------------------------------
* Binaries: 
  Files that can be executed, similar to executables in Windows. They generally 
  reside in the /usr/bin or /usr/sbin directory. 
  Examples: ps, cat, ls, ifconfig, aircrack-ng, and Snort.

* Directory: 
  The same as a folder in Windows, providing a way of organizing files in a 
  hierarchical manner.

* Home: 
  Each user has their own /home directory, where files created by them will be 
  saved by default.

* Kali: 
  A distribution of Linux specifically designed for penetration testing. It has 
  hundreds of tools preinstalled (using Kali 2018.2, first released in April 2018).

* root: 
  The administrator or superuser account designed for a trusted person who can 
  do nearly anything on the system (reconfiguring the system, adding users, 
  changing passwords). Many hacker tools require the root account to give you 
  control over the system.

* Script: 
  A series of commands run in an interpretive environment that converts each 
  line to source code. Run with the bash interpreter or others like Python, 
  Perl, or Ruby. Python is currently the most popular interpreter among hackers.

* Shell: 
  An environment and interpreter for running commands in Linux. The most widely 
  used shell is bash (Bourne-again shell), but other popular shells include the 
  C shell and Z shell.

* Terminal: 
  A command line interface (CLI).

⚠️ Note (Case sensitivity): 
Unlike Windows, the Linux filesystem is case sensitive (e.g., Desktop, desktop, 
and DeskTop represent different files or directories). If you get the error 
"file or directory not found", you probably need to check your case.
